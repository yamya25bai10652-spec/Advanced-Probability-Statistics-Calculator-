Project Overview

This project is a moderate-level command-line (CLI) Probability & Statistics Calculator. It helps users to perform statistical and probability calculations easly from terminal.

It is design to show understanding of maths concepts and programing logic. The application runs fully in CLI which make it suitable for submission.

I really enjoyed building this project and it was fun learning statistics!

Setup & Execution Guide 

This guide will help you run the project even if you dont have prior knowledge.

Step 1: Install Python

Download Python from official website and install it. Make sure you select “Add to PATH” option.

Check installation:

python --version


⸻

Step 2: Clone Repository

git clone https://github.com/{your-username}/probability-statistics-calculator.git
cd probability-statistics-calculator

If git is not installed, you can download zip file also.

⸻

Step 3: Create Virtual Environment

python -m venv venv

Activate it:

Windows:

venv\Scripts\activate

Linux/Mac:

source venv/bin/activate


⸻

Step 4: Install Dependancies

pip install -r requirements.txt


⸻

Step 5: Run Project

python main.py

Now program will start and you can use it

Theory (Formulas)

Mean = Sum of values / total values

Variance = sum of (x - mean)^2 / n

nCr = n! / (r! (n-r)!)
